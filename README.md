Linie 9, kolej 9: 4. Linie 6, kolej 12: 3. Linie 7, kolej 49: 4.

przejdź do gry, a stanie się jasne, jak wygrać w księdze Ra. A jeśli tak się nie stanie, lepiej w ogóle tego nie robić!
